# centos

如果使用yum install xxxx，会找到安装包之后，询问你Is this OK[y/d/N]，需要你手动进行选择。但是如果加上参数-y，就会自动选择y，不需要你再手动选择！ yum比rpm高级一次安一套

yum 包夜 rpm 一次

yum -y install git

# 分支管理
